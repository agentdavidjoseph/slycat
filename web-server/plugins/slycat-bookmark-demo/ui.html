This model demonstrates how Slycat's bookmarking mechanism works. To see it in action, enter some text and click the Save button:

Now, if you mail a link to the current page or bookmark it in your browser, opening the link will display the same text.

Plus, any time you reopen an existing model, your browser automatically remembers the last state.

You can bookmark and share as many different states as you like - each unique state will have a different page URL.

Expert Tip: If you ever want the browser to "forget" the last state and open a model as if for the first time, edit the URI in your browser address bar so it ends with ?bid=clear
